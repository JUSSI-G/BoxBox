Pipeline

Three-file pipeline

One command runs everything.

01
analyser.py
Variance Analyser
Loads pit stop CSVs. Computes pit window errors and execution penalties. Fetches grid positions from Ergast.
→ window_penalty_s
→ exec_penalty_s
→ avg_window_error_laps
→ era_code
02
xgb.py
XGBoost Model
Trains on 1994-2011, tests on 2012-2024. Runs per-era importance analysis. Saves results for the dashboard.
→ f1_xgb_dataset.csv
→ f1_era_importance.json
03
f1.py
Championship Simulator
Zeros variance features per driver-race, keeps grid and era fixed. Flags seasons where perfect strategy would have changed the title.
→ f1_simulation_sweep.csv
